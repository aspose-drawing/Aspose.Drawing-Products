<p align="justify">
 Aspose.Drawing für .NET ist eine plattformübergreifende 2D-Zeichnungs-Engine mit System.Drawing-kompatibler API. Die Zeichnungsbibliothek unterstützt das Rendern von Vektorgrafiken wie Linien, Kurven und Figuren sowie Text in einer Vielzahl von Schriftarten, Größen und Stilen auf Rasterbildern. Bilder können in allen gängigen Grafikformaten gespeichert werden, einschließlich BMP, PNG, JPEG, GIF und TIFF. Aspose.Drawing ist eine reine .NET-Bibliothek und verlässt sich nicht auf eine externe native Rendern-Engine.
</p>
