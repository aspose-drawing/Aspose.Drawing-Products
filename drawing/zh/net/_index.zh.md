<p align="justify">
 Aspose.Drawing for .NET 是一个具有 System.Drawing 兼容 API 的跨平台 2D 绘图引擎。绘图库支持将线、曲线和图形等矢量图形以及各种字体、大小和样式的文本渲染到光栅图像上。图像可以保存为所有常用的图形格式，包括 BMP、PNG、JPEG、GIF 和 TIFF。 Aspose.Drawing 是一个纯 .NET 库，不依赖任何外部原生渲染引擎。
</p>
