<p align="justify">
 Aspose.Drawing for .NET 是一個具有 System.Drawing 兼容 API 的跨平台 2D 繪圖引擎。繪圖庫支持將線、曲線和圖形等矢量圖形以及各種字體、大小和样式的文本渲染到光柵圖像上。圖像可以保存為所有常用的圖形格式，包括 BMP、PNG、JPEG、GIF 和 TIFF。 Aspose.Drawing 是一個純 .NET 庫，不依賴任何外部原生渲染引擎。
</p>
