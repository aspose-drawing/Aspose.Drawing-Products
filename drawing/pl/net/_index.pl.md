<p align="justify">
 Aspose.Drawing dla platformy .NET to wieloplatformowy silnik do rysowania 2D z interfejsem API zgodnym z System.Drawing. Biblioteka rysunków obsługuje renderowanie grafiki wektorowej, takiej jak linie, krzywe i figury, a także tekstu w różnych czcionkach, rozmiarach i stylach na obrazy rastrowe. Obrazy można zapisywać we wszystkich powszechnie używanych formatach graficznych, w tym BMP, PNG, JPEG, GIF i TIFF. Aspose.Drawing to czysta biblioteka .NET i nie opiera się na żadnym zewnętrznym natywnym silniku renderującym.
</p>
